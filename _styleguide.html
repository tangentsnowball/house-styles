Grid

The grid is a responsive percentage based grid that uses the same markup and class names as the bootstrap 2.0 grid. This is intentional to support oscar projects if required. The grid is generated in the file grid.less and contains several variables to easily reconfigure the grid if required, e.g. to have a different number of columns.

It collapses to a single column below a certain screen width. The first element in a row automatically has no left margin. See the grid test file for a full grid layout.

Buttons

Buttons are controlled through the .btn class, with additional classes to control the appearance and size.

Buttons should look exactly the same for any reasonable element type they are applied to, although there seems to be an inherent difficulty in making input elements match the height of the other elements cross-browser. It is therefore recommended that only a and button are used for buttons.

Basic styles:

  • .btn-primary
  • .btn-secondary
  • .btn-info
  • .btn-danger

Additional button styles:

  • .disabled
  • .btn-small
  • .btn-large
  • .btn-block

Tabs require the .js-tabbable class in order to be active, otherwise they simply link to another page.
